# pg-tables-to-zod

A command-line utility and module to turn postgres tables into Zod schemas

## Installation

```bash
bun add pg-tables-to-zod
```

## Usage

```typescript
import { greet } from 'pg-tables-to-zod';

console.log(greet('World')); // Hello, World!
```
